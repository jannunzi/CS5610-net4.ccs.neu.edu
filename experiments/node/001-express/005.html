Responding with JSON from NodeJS Express

Introduction

Increasingly we use JSON as the standard format of data criscrossing the Internet today. Express and NodeJS make it easy to send and receive JSON to and fro between a client (browser) and a server (NodeJS). Here we show how to respond with JSON from a client HTTP request.

Hello JSON

To respond with JSON, it is as simple as using the res.json() method to respond with JSON. Let's respond with a simple "Hello JSON" message from the server. Add the following lines to the previous example. We are binding an HTTP URL request for "/hello" to a JSON object response. 

app.get('/hello', function(req, res){
    var jsonObject = {
        message: "Hello JSON"
    };
    res.json(jsonObject);
});

You should now have the following content in server.js 

var express = require('express');
var app = express();

app.get('/', function(req, res){
  res.send('Hello World');
});

app.get('/hello', function(req, res){
    var jsonObject = {
        message: "Hello JSON"
    };
    res.json(jsonObject);
});

var server = app.listen(3000, function() {
    console.log('Listening on port %d', server.address().port);
});

Try hitting the service by pointing your browser to http://localhost:3000/hello. The server should respond with 

{
    message: "Hello JSON"
}
