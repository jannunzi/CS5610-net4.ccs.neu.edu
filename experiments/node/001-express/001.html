Express

Introduction

NodeJS has many great packages available through NPM or (Node Package Manager). You can install new packages by using the npm command installed when you installed NodeJS.

One of the more popular packages is Express. Express allows creating servers and Web Services in NodeJS very easy. First we need to install express in our NodeJS application.

Install the Express Node Module

Use npm to install express in your application. Create a directory where your application will exist, change to that directory and then execute the following in your command line: 

npm install express
Npm will create a node_modules directory and download all the necessary libraries to that directory.

Create a Hello World Server

Now let's create a simple hello world application. Create a file called server.js, and load the express module and create an express application by typing the following in the file. The name of the file does not matter. 

var express = require('express');
var app = express();

Now let's bind an HTTP request of the root of the application Web context "/" to an HTTP response. Keep typing in the server.js file: 

app.get('/', function(req, res){
    res.send('Hello World');
});

Finally have the express application listen for HTTP requests at port 3000 (a common PORT number). The anonymous function used in listen() just notifies that the server is ready and listening. Type the following in server.js 

var server = app.listen(3000, function() {
    console.log('Listening on port %d', server.address().port);
});

You should have the following server.js content 

var express = require('express');
var app = express();

app.get('/', function(req, res){
  res.send('Hello World');
});

var server = app.listen(3000, function() {
    console.log('Listening on port %d', server.address().port);
});

Let's now run the server from the command line as follows 

> node server.js
Listening on port 3000

Now that the server is listening to port 3000, try hitting by pointing a browser to http://localhost:3000. The server should respond with "Hello World"
