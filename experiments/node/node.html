NodeJS

Introduction

NodeJS allows executing JavaScript on a desktop. With it's extensive library of really useful scripts, nodejs is quickly becoming great platform for developing very powerful server applications. Read more about nodejs from it's main site at http://nodejs.org/

Download and Install NodeJS

NodeJS can be downloaded and installed in most operating systems . The installation installs several executables in your local hard drive. In Windows environments, it might be in C:\Program Files (x86)\nodejs. The two executables we care about are node and npm. Add the directory to your path so that you can execute these from anywhere in the command line.

Hello World NodeJS

To create a NodeJS program, simple create a JavaScript file with valid JavaScript code. The only limitation of the JavaScript code is that it is not running in a browser, it's running in your desktop. Which is not a limitation at all. Since it's running on your desktop, you can do all sorts of things you could not do before. You can access the file system, you can connect to databases, you can do network connections that would be allowed from a browser.

Let's create a Hello World program. In some directory, create a file called hello.js and type the following in the file: 

console.log("Hello World!");
Then from the command line, change to the directory where the file is located and type the following at the command line to run your program: 
> node hello.js
Hello World!
Note that the program wrote to the console. Congrats, you just wrote your first NodeJS program.
