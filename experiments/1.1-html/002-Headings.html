HTML Heading Elements

Use HTML Heading Elements to label your content into logical sections. There are six heading elements. The six heading elements, H1 through H6, denote section headings. Although the order and occurrence of headings is not constrained by the HTML DTD, documents should not skip levels (for example, from H1 to H3)

Only Use One H1 Element

It is recommended to use one H1 element on any page denoting the top level section. Subsections are denoted with elements H1 through H6. Generally

Only Go Up to H3

Although there are heading elements H1 through H6, in general only H1 through H3 are used.

Use Paragraph Elements to Break Up Text Into Paragraphs

HTML ignores white spaces such as line breaks, multiple spaces, and tabs. They are all rendered as single spaces, so even if you format your text with white spaces, the formatting will be lost when rendered in a browser. White spaces is useful to make the HTML code readable for editing, but it is ignored for rendering. Instead, use HTML layout formatting elements such as heading elements and paragraph elements to format the page.
