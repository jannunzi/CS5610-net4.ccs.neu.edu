Select by Id, Class, or Element Type Experiment

This is the only existing content whose ID attribute is 'selector'

This is one of 2 existing contents whose CLASS attribute is 'selector'

This is the other existing content whose CLASS attribute is 'selector'

Documentation

This experiment illustrates the various ways jQuery can refer to different parts of the DOM. jQuery uses CSS3 selector rules to refer to elements in the DOM. Given the following HTML: 
	<h2 id="selector">This is the only existing content in whose ID attribute is 'selector'</h2>
	<h2 class="selector">This is one of 2 existing contents whose CLASS attribute is 'selector'</h2>
	<p class="selector">This is the other existing content whose CLASS attribute is 'selector'</h2>
jQuery can refer to these elements using the following selectors: 
	$("#selector") // refers to <h2 id="selector">
	$(".selector") // refers to all elements with CLASS='selector', i.e., <h2 class="selector"> and <p class="selector">
	$("h2") // refers to all <h2> elements, i.e., <h2 id="selector"> and <h2 class="selector">
The script goes on to append some content to each of the elements: 
	$("#selector").append(" Add this content to the only element with ID attribute set to 'selector'");
	$(".selector").append(" Add this content to all elements with CLASS attribute set to 'selector'");
	$("h2").append(" Add this content to all elements of type H2");
